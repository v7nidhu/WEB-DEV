What is display

  • element generates block/inline box
  • starts on newline/flow with text
    • width,height,margin and layout

Block

  • takes full available width
  • starts own new line
  • respects al model props
  • ex: div,p,h1-h6,section

Inline

  • Sticks to content width only
  • flows within surronding text
  • ignores width,height,top/bottom margin,padding
  • ex: span,a,strong,em

Inline-Block

  • flows inline like text
  • but respects with width and height
  • top/bottom margin,padding
  • ex: span

divison block

  • flows in a division
  • but respects with width and height
  • top/bottom margin,padding
  • ex: span
